# Casa Doña Rosa Web

Sitio web profesional para Casa Doña Rosa en Soroa, Cuba.

## Incluye

- Landing page moderna y responsive
- Galería con fotos reales
- Reserva directa por WhatsApp
- Botón de reserva por Airbnb
- Contactos de Milagro Real y Ricardo Delgado
- Diseño listo para publicar en Vercel, Netlify o Bolt

## Cómo correr localmente

```bash
npm install
npm run dev
```

## Cómo publicar en Vercel

1. Sube esta carpeta a GitHub.
2. Entra a Vercel.
3. Importa el repositorio.
4. Build command: `npm run build`
5. Output directory: `dist`
6. Deploy.
